# Sales-Data-Analysis-and-Visualization

This project analyzes sales data to understand product performance,
monthly trends, and regional sales distribution.

## Features
- Data cleaning
- Sales analysis
- Bar chart visualization
- Line chart trend analysis
- Pie chart distribution

## Technologies Used
Python
Pandas
Matplotlib

## Project Structure
data/
visualizations/
report/

## Insights
Electronics category shows highest sales while some categories show lower demand.
